Installing for Ubuntu 14.04 (Trusty Tahr)

You can easily try us out via our PPA:

sudo apt-add-repository ppa:trapni/xzero
sudo apt-get update
sudo apt-get install x0d
sudo cp /etc/xzero/x0d.conf-dist /etc/xzero/x0d.conf
sudo initctl start x0d

Installing for Docker

This is how it could look like when running Xzero via docker abstraction.

docker pull trapni/xzero-git
docker run -i -p 80:80 -v /etc/xzero:/etc/xzero -v /var/www:/var/www trapni/xzero-git
