Xzero HTTP Web Server & Framework

About Xzero

It's not just a software!

Xzero is about Usability

Xzero features a very powerful configuration language, Flow, that makes your x0d HTTP server highly flexible.
The configuration language as well as the unix tools and configuration files all come with their dedicated UNIX manual pages, so that you can read up documentation from everywhere.

  • Very flexible and easily maintainable configuration language: Flow
  • Extend the core functionality by loading of custom plugins
  • Configurationless instant-mode for quickly serving HTML apps and/or directories.
  • Sane and secure defaults
  • Minimal dependencies

Xzero is about Scalability

Xzero is designed from ground up with scalability, performance and low latency in mind, hence, this is why this project has it's name. Even though, the bar is raised very high, we still hold tight on our origin goals.

  • Highly Multithreaded, including CPU core cache level optimizations
  • Zero Copy Networking, making use of the operating systems kernel APIs as much as possible
  • Non-Blocking I/O, in order to achieve literally zero latency

Xzero is about Extensibility

Xzero is devided into multiple smaller components. It features a very slim core and lets you build your application ontop of it. x0d, the main HTTP Web Server is the main application build with the Xzero Framework, it allows you to extend the core by various standard plugins as well as third party extensions.

  • SSL support, including SNI extension
  • Dynamic Content Compression
  • Advanced Load Balancing, with business grade features
  • Basic Authentication
  • Directory Listing
  • User Directories
  • ... and much more ...
Follow up the Plugin Documentation Index to get a full list.
